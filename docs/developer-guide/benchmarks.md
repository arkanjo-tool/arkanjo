# Evaluator Benchmark

Run and inspect ArKanjo benchmark evaluations.

## Quick Start

### Requirements (BigCloneBench)

* Python
* Java
* ArKanjo installed and available in `PATH`
* BigCloneBench database [^2]
* IJaDataset [^3]
* H2 JDBC driver [^4]

```bash
pip install -e ".[benchmark]"
```
