# Patch History using ArKanjo - MAC0470/5856 (2025)

This file records all patches submitted by students of the course **MAC0470/5856 - Free Software Development (2025)** using the **ArKanjo** tool, targeting the Linux kernel community.

It aims to:

* Provide real examples of successful contributions
* Help new contributors understand the review process

## Participant Observation Experiment

- Subsytem: `amd-gfx`
- Patch: https://lore.kernel.org/all/20250225015532.303032-1-luanicaro@usp.br/
- Accepted on V4

This patch removes duplicated helper functions across multiple `command_table_helper_*` files in the AMD display driver.

The following functions were centralized into a common helper file:

* `phy_id_to_atom()`
* `clock_source_id_to_atom_phy_clk_src_id()`
* `engine_bp_to_atom()`

All duplicated implementations of these functions were removed from architecture-specific files (`dce60`, `dce80`, `dce110`, `dce112`, etc.), and the shared versions are now declared in `command_table_helper.h` and implemented in `command_table_helper.c`.

#### Group 8

- Patch: https://lore.kernel.org/all/20250614124401.32b17eda@jic23-huawei/

#### Group 11

- Subsystem: `amd-gfx`

This patch removes duplicated implementations of `hpd_ack` across multiple AMD DC IRQ service files by centralizing the logic in `irq_service.c`.

Two shared helpers, `hpd0_ack` and `hpd1_ack`, are introduced, and all callers are updated to use them accordingly. The required register field definitions are also added.
